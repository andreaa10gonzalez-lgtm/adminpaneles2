# Casino Panel

Sistema de gestión operativa para paneles de casino online.

## Setup

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Subí este repo a GitHub
2. Entrá a vercel.com → New Project → importá el repo
3. Deploy automático ✅

## Acceso inicial

- **Super Admin:** `admin` / `admin2024`

Desde el panel de admin creás los accesos para cada cliente.
